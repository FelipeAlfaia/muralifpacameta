🔥 Configurar Firebase

Para que os dados fiquem salvos no servidor e compartilhados entre todos os alunos, você precisa criar um projeto gratuito no Firebase e colar as credenciais abaixo.

Passo 1: Acesse console.firebase.google.com → "Adicionar projeto" → dê um nome (ex: ifpa-cameta) → clique em Continuar até criar.

Passo 2: Dentro do projeto, clique em Firestore Database (menu lateral) → "Criar banco de dados" → modo Produção → escolha a região southamerica-east1.

Passo 3: Ainda no Firestore, vá em Regras e substitua o conteúdo por:

rules_version = '2'; service cloud.firestore { match /databases/{database}/documents { match /opinioes/{doc} { allow read: if true; allow create: if true; allow update: if request.resource.data.diff(resource.data).affectedKeys() .hasOnly(['likes']); } match /eventos/{doc} { allow read: if true; allow write: if false; } match /merenda/{doc} { allow read: if true; allow write: if false; } } }

⚠ Eventos e merenda só podem ser editados via painel do Firebase (pelos servidores). O PIN de admin neste site libera a interface — as regras acima são a segurança real.

Passo 4: Vá em ⚙ Configurações do projeto → "Seus aplicativos" → clique em </> (Web) → registre o app → copie o objeto firebaseConfig e cole aqui:

💡 No Firebase: ⚙ Configurações do projeto → Seus aplicativos → selecione o app web → copie todo o bloco firebaseConfig
